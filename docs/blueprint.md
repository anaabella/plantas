# **App Name**: PlantPal PWA 

## Core Features:

- Plant Acquisition Tracking: Record details about how each plant was acquired (purchased, gifted, traded), along with the price and reason for trade.
- Plant Details: Allow users to track root, leaf, or clipping information for each plant.
- Categorized Grid Display: Display plants in a grid format with image, name, and acquisition date.
- Mortality Tracking: Enable marking plants as deceased to track plant lifecycles.

## Style Guidelines:

- Primary color: Soft, natural green (#A7D1AB) to evoke nature and growth.
- Background color: Light, desaturated beige (#F5F5DC) for a calm and earthy feel.
- Accent color: Muted terracotta (#E07A5F) to add warmth and highlight key information.
- Body and headline font: 'Alegreya', a serif font, giving an elegant feel.
- Use simple, line-based icons to represent plant types and actions (e.g., watering can, sun).
- Grid-based layout for plant display, optimized for various screen sizes.
- Subtle fade-in animations when displaying plant details or updating information.
